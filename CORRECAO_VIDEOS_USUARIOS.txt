OPÇÃO 1: Usar deploy automático (Recomendado)

❓ TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Problema: "Endpoint não encontrado"
Solução: Verificar se API está rodando em http://127.0.0.1:8001

Problema: "URL do YouTube inválida"
Solução: Certifique-se que URL contém "youtube" ou "youtu.be"

Problema: "Video ID inválido"
Solução: URL deve ter formato padrão: https://youtube.com/watch?v=ID

Problema: yt-dlp timeout
Solução: 
  • Verificar se yt-dlp está instalado: yt-dlp --version
  • Aumentar timeout em extract.php ou api.py
  • Atualizar yt-dlp: pip install --upgrade yt-dlp

Problema: "Using generic URL"
Solução: yt-dlp falhou, mas retorna fallback genérico
