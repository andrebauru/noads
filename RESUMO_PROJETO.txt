🎬 DESCRIÇÃO DO PROJETO
═══════════════════════════════════════════════════════════════════════════

YouTube without Ads + Chromecast with Auto-Discovery

Full Description:
"Noads is a modern web application that allows you to watch YouTube videos 
without ads, download them as MP4/MP3, and seamlessly stream to your 
Chromecast devices with automatic discovery. Features include robust Cast API 
v3.0, periodic device detection, auto-transmission on playlists, and 
background playback. Built with HTML5, Python Flask, and modern JavaScript."

📊 CONTEÚDO DO PROJETO
═══════════════════════════════════════════════════════════════════════════

🎬 FEATURES
  ✅ Reprodução YouTube sem anúncios
  ✅ Detecção automática de Chromecast
  ✅ Auto-discovery periódico (5 segundos)
  ✅ Transmissão automática em playlists
  ✅ Download MP4 (video) + MP3 (áudio)
  ✅ Suporte a múltiplos Chromecasts
  ✅ Video em background (não pausa ao mudar aba)
  ✅ Interface responsiva
  ✅ Logs detalhados para debug
  ✅ Tratamento robusto de erros

💻 TECNOLOGIAS
  • Frontend: HTML5 + CSS3 + JavaScript Vanilla
  • Backend: Python 3.6+ (Flask)
  • Backend: PHP 7.2+
  • Video: yt-dlp
  • Cast: Google Chrome.cast API v3.0
  • Storage: LocalStorage + Server Cache
