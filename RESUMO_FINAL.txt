❌ PROBLEMA 2: "API não responde (Connection refused)"
   SOLUÇÃO:
   $ sudo systemctl restart noads
   $ sudo netstat -tuln | grep 8001
   (deve mostrar LISTEN)

❌ PROBLEMA 3: "Repositório desatualizado (versão antiga)"
   SOLUÇÃO:
   $ cd /var/www/noads
   $ git pull origin main
   $ git log --oneline -1
   (deve mostrar commit recente com "Fix: Remover restricao")

❌ PROBLEMA 4: "Permissões erradas"
   SOLUÇÃO:
   $ sudo chown -R www-data:www-data /var/www/noads
   $ sudo chmod -R 755 /var/www/noads
   $ sudo systemctl restart noads

❌ PROBLEMA 5: "NGINX/Apache proxy não configurado"
   SOLUÇÃO: Ver DEPLOY_ATUALIZADO.txt seção "CONFIGURAR NGINX/APACHE"
   E depois: $ sudo systemctl restart nginx (ou apache2)
