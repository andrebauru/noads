PASSO 3: AUTENTICAR COM GITHUB
────────────────────────────────────────────────────────────────────────────

Se pedir senha/autenticação:

OPÇÃO A: GitHub CLI (Recomendado)
──────────────────────────────────
```powershell
# Se tiver gh instalado:
gh auth login
# Siga as instruções
```

OPÇÃO B: Token de Acesso Pessoal
──────────────────────────────────
1. Acesse: https://github.com/settings/tokens
2. Clique "Generate new token"
3. Selecione "repo" (full control of private repositories)
4. Clique "Generate token"
5. COPIE o token (será visível apenas uma vez)
6. Use o token como senha ao fazer push:

```powershell
git push -u origin main
# Usuário: seu_usuario_github
# Senha: cole_o_token_aqui
```
